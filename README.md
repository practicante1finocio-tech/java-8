# Java 8 Functionalities

This repository contains a complete set of Java 8 functionalities, exercises and examples implemented as part of a project. It is designed for learning and practicing core Java 8 concepts.

## Features

- Implementation of Java 8 features:
  
  - Lambdas
  - Streams
  - Optional
  - Method references
  - Default and static methods
  - CompletableFuture
  - Date and Time API
 
- Practical exercises for each feature
- Example code demonstrating best practices

## Requirements

- Java 8 or higher
- Gradle
